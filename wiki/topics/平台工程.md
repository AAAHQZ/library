# 平台工程

> 平台工程（Platform Engineering）是通过构建内部开发平台来提升研发效能的工程实践，核心是让开发者自助获取基础设施能力。

## 核心概念

- [[2026-05-05-云原生基础设施SRE落地项目从平台建设到稳定性工程闭环|sources/2026-05-05-云原生基础设施SRE落地项目从平台建设到稳定性工程闭环]] — Site Reliability Engineering，通过工程化和自动化手段提升系统可靠性
- GitOps — 以 Git 为单一事实来源的运维方法论
- 可观测性 — 指标（Metrics）、日志（Logs）、链路追踪（Traces）三大支柱
- SLI/SLO — 服务等级指标与服务等级目标

## 主要实践

### 环境基线管理
B 站等企业的系统环境基线管理实践，通过声明式配置管理实现环境一致性。

### 可观测性建设
云原生环境下的 SRE 落地，强调从平台建设到稳定性工程闭环。

### 平台即产品
平台工程团队将内部平台视为产品运营，以开发者体验为核心进行迭代。
