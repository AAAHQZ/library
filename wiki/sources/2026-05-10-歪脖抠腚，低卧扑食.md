# 歪脖抠腚，低卧扑食

> Vibe Coding（歪脖抠腚）的音译开篇——DevOps 是 Vibe Coding 时代最后的护栏。一个六年 DevOps 老兵的个人号发刊词。

## 核心观点

1. **"歪脖抠腚"是 Vibe Coding 的音译**——歪着头挠着屁股，AI 就把活干了。爽是真的爽，但混乱也是真的混乱。

2. **"低卧扑食"是 DevOps**——趴低了干活，往实里扑，守住工程纪律的底线。DevOps 的工程纪律（可重复的构建、可观测的部署、可追溯的变更）在 AI 时代不是过时了，而是比任何时候都重要。

3. **核心论点**：DevOps 是 Vibe Coding 时代最后的护栏。AI 把开发速度干到 10 倍，但如果交付还是慢得不行、线上事故频发，10 倍速度只会制造 100 倍的混乱。

4. **内容方向**：Harness Engineering（如何驾驭大模型）、Agentic CI/CD、平台工程、AI 编程实践与工程纪律、工程管理。

5. **从 Vibe Coding 到 Harness Engineering 的转向本质**：把 Agent 当"牲口"管，不是当"宠物"养——可替代、可审计、有边界、可观测。
