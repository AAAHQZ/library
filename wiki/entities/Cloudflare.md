# Cloudflare

> 全球领先的 CDN 和安全服务提供商，以慷慨免费层闻名。

## 在 Agent 时代的意义

Cloudflare 的免费层承载了大量 AI Agent 和爬虫流量。以 Pigsty.io 为例，每月过亿请求量全部经过 Cloudflare 而无需付费，成为"Agent 时代内容分发基础设施"的代表性案例。
