# Wardley Map

Wardley Map 是一种战略可视化工具，用于映射价值链中各组件的演进位置。

## 轴向

| 轴 | 说明 |
|---|---|
| **Y 轴** | 可见性：从用户（顶部）到基础设施（底部） |
| **X 轴** | 演进：从新颖（左侧）到商品化（右侧） |

## 演进阶段

```
[构建] → [自定义] → [产品] → [商品] → [ Commoditized ]
  ↑                                      ↑
 novelty                              commodity
```

- **Novelty (新颖)**: 早期探索，定制解决方案
- **Custom Built (定制)**: 差异化，特定需求
- **Product (产品)**: 标准化，市场选择
- **Commodity (商品化)**: 基础设施，商品服务

## 示例结构

```
                                    用户
                                     ↑
                              [ 需求定义 ]
                                     |
                              [ 用户界面 ]
                                     |
                          [ 业务逻辑 ]
                                     |
                        [ 数据存储 ]
                                     |
                     [ 基础设施 ]
                                     |
                              [ 云服务 ]
```

## 图例
