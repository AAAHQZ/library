# Superpowers

> Jesse Vincent团队开发的Claude Code Markdown skill集合，把高质量软件工程工作流编码成agent能严格遵守的skill

## 简介

14个skill分三组：计划与思考、编码与调试、审查与验证。两个反直觉设计：(1)独立reviewer通道——新开上下文审查，避免同一上下文自我审查；(2)verification-before-completion——没有证据的完成等于没完成。局限：使用门槛高、Spark阶段薄（只靠brainstorming）、项目级状态无归宿、棕地缺记忆、质量门禁规模化难。
