# 上下文供给（Context Supply）

> 企业级 Agent 落地的核心瓶颈，指为 Agent 构建多源、实时、可信上下文的能力。被认为是比模型能力更关键的分水岭。

## 简介

上下文供给（Context Supply）是环境工程范式的核心关注点。在企业级 Agent 落地过程中，即使模型能力再强，如果 Agent 无法持续、低成本、可信地接入真实业务世界的数据，也难以做出合理决策。

## 关键信息

- **类型**：概念
- **领域**：AI Agent 工程、企业级 AI
- **相关概念**：[[环境工程]]、[[EventHouse]]、[[统一Catalog]]、[[知识对账]]

## 详细内容

### 上下文供给的五个关键判断

1. **信息完备性是前提**：先让 Agent 看见真实业务世界。AI Coding 成功的原因在于软件工程环境天然数字化
2. **信息不是越多越好**：需要统一 Catalog 作为"图书馆馆藏目录"而非整座图书馆
3. **知识不是信息囤积**：基于 DIKW 模型，需要可审查、可解释的知识对账机制
4. **每一次知识迭代都是生产级变更**：需要制品化发布和变更治理
5. **简单与可靠是普惠的入场券**：需要像"接电"一样的标准化接入体验
