# Agentic项目管理

> AI时代项目管理新范式——从"人驾驭AI管项目"走向"AI主导、人打辅助"，最终愿景是AI完全接管项目管理。

## 简介

Agentic项目管理是小红书PMO团队提出的"AI时代项目管理新范式"愿景。核心判断：AI是一种新生产力，会越来越聪明，最终必然走向AI完全接管项目。当前正处于"人驾驭AI管项目"阶段。

## 详细内容

### 四阶段演进路径

| 阶段 | 描述 |
|------|------|
| 人管项目 | 依赖PMO人工盯细节 |
| 人驾驭AI管项目 | 当前阶段，AI执行人规划的routine工作 |
| AI主导、人打辅助 | AI主导决策，人提供辅助判断 |
| AI完全接管项目 | 愿景，AI全面负责项目管理 |

### 企业项目管理演进阶段

过去企业落地项目管理通常经历：
1. **人治**（依赖PMO人工盯细节）
2. **机制化**（流程机制落地、建立项目管理体系）
3. **工具化**（以项目管理平台为核心）
4. **AI化**（AI Agent接管管理）

### 两条关键判断

1. **AI是新生产力**：应该用它把routine工作（拉群、催更、总结进展、识别todo、写纪要等）真正接管掉，把人的精力留给真正需要决策的地方——目标、阵型、资源。

2. **AI会越来越聪明**：随着大模型和Agent工程的发展，项目管理必将经历四阶段演进，当前正处"人驾驭AI管项目"阶段。

### 7条实践经验

1. 不要等各种设施齐了才开始
2. 评测集是Agent的"单元测试"
3. 原子Agent必须能自我闭环
4. 记忆结构化是更智能的底座
5. 领域Agent = 领域Source of Truth
6. 项目主数据先于场景扩展
7. AI时代，人人都可以是Builder
